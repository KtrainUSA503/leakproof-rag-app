Typical Usage Pattern
---------------------

1. First Time Setup:
   pip install -r requirements.txt
   export OPENAI_API_KEY='your-key'

2. Initialize System:
   from leakproof_rag import LeakProofRAG
   rag = LeakProofRAG()

3. Load Document:
   rag.load_document("leakproof_drive.pdf")
   rag.create_embeddings()

4. Query:
   result = rag.query("Your question here")
   print(result['answer'])

5. Optional - Save for later:
   rag.save_index("my_index.json")

6. Next time - Load saved index:
   rag = LeakProofRAG()
   rag.load_index("my_index.json")
   result = rag.query("Another question")
